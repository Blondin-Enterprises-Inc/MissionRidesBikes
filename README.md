# My Static Site

This is a simple static website hosted on GitHub (or WHC).

## Structure

- `index.html` — homepage
- `css/styles.css` — basic styling
- `js/script.js` — basic script

## How to Use

1. Clone this repo
2. Open `index.html` in a browser
3. Push to GitHub
4. (Optional) Deploy with GitHub Pages or to your VPS/WHC server
